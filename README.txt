class.py -> has all classes used, including Transaction and Date
data.py -> has all data used, including list of transactions and dictionaries
driver.py -> the python program to run
main.py -> main python code containing various functions

operations.txt -> text files with operations to filter data
    "month" command finds total expense and count about a specific category in a specific month
    "compare" command outputs a string based on a comparison between a given budget and monthly expense
    "reset" command removes all the data back to noting
    "add_transaction" command provides user the ability to add transactions
    "set_budget" command assigns a user inputted value to a monthly budget
    there is also code to skip blank lines and display an error message whenever an operation is not inputted correctly
